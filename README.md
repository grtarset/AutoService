# AutoService

Десктопна програма для автосервісу на PySide6 та SQLite.

## Можливості

- створення актів виконаних робіт;
- окремі довідники клієнтів і автомобілів;
- журнал клієнтів із редагуванням карток;
- кілька телефонів і кілька автомобілів в одній картці клієнта;
- автозаповнення клієнта/авто за ПІБ, телефоном, VIN або держномером;
- журнал актів із пошуком;
- матеріали, запчастини, послуги та підсумки;
- експорт акту в PDF;
- автоматична міграція старої `database/autoservice.db` у робочу базу користувача.

## Дані

Робоча база зберігається в `%APPDATA%\AutoService\autoservice.db`.
Якщо такої бази ще немає, програма копіює стару базу з `database/autoservice.db` і мігрує її до нової схеми.

Для тестів або переносної інсталяції можна задати змінну середовища `AUTOSERVICE_DATA_DIR`.

PDF використовує bundled DejaVu Sans 2.37 зі шрифтів у `fonts/`; ліцензія лежить у `fonts/LICENSE-DejaVu.txt`.

## Збірка

```powershell
.\build.ps1
```

Скрипт перед збіркою закриває запущений `AutoService.exe`, очищає попередні `build\AutoService` і `dist\AutoService` з кількома повторними спробами та запускає PyInstaller через `AutoService.spec`.

Якщо потрібно використати конкретний Python:

```powershell
.\build.ps1 -Python "py"
```
